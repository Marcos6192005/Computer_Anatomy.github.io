Hardware
Image Name Concepto Concept
CPU (Central Processing Unit) El cerebro de la computadora que ejecuta los programas y las tareas The brain of the computer that executes programs and tasks
RAM (Random Access Memory) La memoria temporal que almacena los datos que la CPU está utilizando actualmente The temporary memory that stores the data that the CPU is currently using
Hard drive El almacenamiento permanente de la computadora que almacena los archivos, el sistema operativo y otros datos. The computer's permanent storage that stores files, the operating system, and other data
Motherboard La placa base de la computadora que conecta todos los componentes The computer's motherboard that connects all the components
Video card La tarjeta que genera la imagen que se muestra en el monitor The card that generates the image that is displayed on the monitor
Sound card La tarjeta que genera el sonido que se escucha a través de los altavoces o auriculares The card that generates the sound that is heard through speakers or headphones
Monitor La pantalla que muestra la imagen generada por la tarjeta de video The screen that displays the image generated by the video card
Keyboard El dispositivo que se utiliza para ingresar texto y datos a la computadora The device that is used to input text and data into the computer
Mouse El dispositivo que se utiliza para controlar el cursor en la pantalla The device that is used to control the cursor on the screen
Printer El dispositivo que imprime documentos, imágenes y otros materiales The device that prints documents, images, and other materials
Scanner El dispositivo que escanea imágenes y documentos para que se puedan guardar en la computadora The device that scans images and documents so that they can be saved on the computer
Power supply La fuente de alimentación que proporciona energía a los componentes de la computadora The power supply that provides power to the computer's components
USB device El dispositivo que se conecta a la computadora a través de un puerto USB The device that connects to the computer through a USB port
CD/DVD drive El dispositivo que lee y escribe discos CD y DVD The device that reads and writes CD and DVD discs
GPU (Graphics Processing Unit) La tarjeta que genera gráficos en 3D y otros efectos visuales The card that generates 3D graphics and other visual effects
Network card La tarjeta que conecta la computadora a una red de área local (LAN) o a Internet The card that connects the computer to a local area network (LAN) or to the Internet
Router El dispositivo que conecta varias redes entre sí The device that connects multiple networks together
Switch El dispositivo que conecta varios dispositivos en una red The device that connects multiple devices in a network
Hub El dispositivo que conecta varios dispositivos a un puerto USB The device that connects multiple devices to a USB port
Bluetooth device El dispositivo que se conecta a otros dispositivos a través de la tecnología Bluetooth The device that connects to other devices through Bluetooth technology


Sofware
Image Name Concepto Concept
Operating system Un software que controla los recursos de hardware y software de una computadora A software that controls the hardware and software resources of a computer
Application Un programa informático diseñado para realizar una tarea específica A software program designed to perform a specific task
Program Un conjunto de instrucciones que se ejecutan en una computadora A set of instructions that are executed by a computer
Database management system (DBMS) Un software que permite almacenar y gestionar datos en una base de datos A software that allows data to be stored and managed in a database
Programming language Un lenguaje formal que se utiliza para crear programas informáticos A formal language that is used to create computer programs
Compiler Un programa informático que convierte código fuente en código objeto A software program that converts source code into object code
Debugger Un programa informático que ayuda a encontrar y corregir errores en programas informáticos A software program that helps find and correct errors in computer programs
Integrated development environment (IDE) Un entorno de software que proporciona herramientas para el desarrollo de programas informáticos A software environment that provides tools for the development of computer programs
Web browser Un software que permite visualizar páginas web en una computadora A software that allows web pages to be viewed on a computer
Antivirus Un programa informático que protege los sistemas informáticos de los virus informáticos A software program that protects computer systems from computer viruses
Firewall Un dispositivo o software que controla el tráfico de red y protege los sistemas informáticos de los ataques externos A device or software that controls network traffic and protects computer systems from external attacks
Version control system Un sistema de software que permite rastrear los cambios en archivos y código fuente A software system that allows tracking changes to files and source code
Productivity software Un software que ayuda a las personas a realizar tareas cotidianas Software that helps people perform everyday tasks
Content management system (CMS) Un software que permite crear y gestionar sitios web A software that allows websites to be created and managed
Office suite Un conjunto de programas informáticos que se utilizan para tareas de oficina, como procesamiento de textos, hojas de cálculo y presentaciones A set of software programs that are used for office tasks such as word processing, spreadsheets, and presentations
Graphic design software Un software que se utiliza para crear imágenes, gráficos y diseños Software that is used to create images, graphics, and design
Video editing software Un software que se utiliza para editar videos Software that is used to edit videos
Audio editing software Un software que se utiliza para editar audio Software that is used to edit audio
3D modeling software Un software que se utiliza para crear modelos 3D Software that is used to create 3D models
Project management system Un software que ayuda a las personas a planificar, coordinar y gestionar proyectos Software that helps people plan, coordinate, and manage projects


Redes y comunicaciones
Image Name Concepto Concept
Internet Una red global de computadoras que están conectadas entre sí A global network of computers that are connected to each other
Protocol Un conjunto de reglas que definen cómo se comunican los dispositivos en una red A set of rules that define how devices communicate in a network
IP (Internet Protocol) Un número único que identifica a cada dispositivo en una red A unique number that identifies each device on a network
LAN (Local Area Network) Una red de computadoras que se encuentran en un área geográfica pequeña, como una oficina o un hogar A network of computers that are located in a small geographic area, such as an office or home
WAN (Wide Area Network) Una red de computadoras que se extiende a una gran área geográfica, como un país o un continente A network of computers that spans a large geographic area, such as a country or a continent
Wi-Fi Una tecnología inalámbrica que permite a los dispositivos conectarse a Internet sin cables A wireless technology that allows devices to connect to the Internet without wires
Ethernet Una tecnología cableada que permite a los dispositivos conectarse a Internet a través de cables A wired technology that allows devices to connect to the Internet through cables
Wireless router Un dispositivo que conecta dispositivos inalámbricos a una red A device that connects wireless devices to a network
VPN (Virtual Private Network) Una red privada que se crea sobre una red pública, como Internet A private network that is created over a public network, such as the Internet
DHCP (Dynamic Host Configuration Protocol) Un protocolo que asigna direcciones IP a los dispositivos en una red A protocol that assigns IP addresses to devices on a network
DNS (Domain Name System) Un sistema que traduce nombres de dominio en direcciones IP A system that translates domain names into IP addresses
Network firewall Un dispositivo que protege una red de amenazas externas A device that protects a network from external threats
Modem Un dispositivo que convierte las señales analógicas en digitales y viceversa A device that converts analog signals into digital signals and vice versa
Server Un dispositivo que proporciona recursos a otros dispositivos en una red A device that provides resources to other devices on a network
Client Un dispositivo que consume recursos de un servidor A device that consumes resources from a server
HTTP (Hypertext Transfer Protocol) Un protocolo que se utiliza para transferir datos en la World Wide Web A protocol that is used to transfer data on the World Wide Web
HTTPS (Hypertext Transfer Protocol Secure) Una versión segura de HTTP que utiliza el cifrado para proteger los datos A secure version of HTTP that uses encryption to protect data.
FTP (File Transfer Protocol) Un protocolo que se utiliza para transferir archivos entre dispositivos A protocol that is used to transfer files between devices.
Email Un servicio que permite a los usuarios enviar y recibir mensajes electrónicos A service that allows users to send and receive electronic messages
VoIP (Voice over Internet Protocol) Una tecnología que permite realizar llamadas telefónicas a través de Internet A technology that allows phone calls to be made over the Internet.


Seguridad
Image Name Concepto Concept
Cybersecurity La práctica de proteger sistemas, redes y datos de ataques cibernéticos The practice of protecting systems, networks, and data from cyberattacks
Password Una cadena de caracteres secreta que se utiliza para autenticar a un usuario A secret string of characters used to authenticate a user
Authentication El proceso de verificar la identidad de un usuario The process of verifying the identity of a user
LAN (Local Area Network) Una red de computadoras que se encuentran en un área geográfica pequeña, como una oficina o un hogar A network of computers that are located in a small geographic area, such as an office or home.
Encryption El proceso de convertir datos en un formato ilegible sin una clave The process of converting data into an unreadable format without a key
Biometrics La identificación de un individuo basándose en sus características físicas o conductuales The identification of an individual based on their physical or behavioral characteristics.
Vulnerability Un punto débil en un sistema informático o red que puede ser explotado por un atacante A weakness in a computer system or network that can be exploited by an attacker
Malware Software malicioso diseñado para dañar un sistema informático o red Malicious software designed to harm a computer system or network
Spyware Software malicioso que recopila información sobre el usuario de un sistema informático o red Malicious software that collects information about the user of a computer system or network
Phishing Un tipo de ataque cibernético en el que los atacantes intentan engañar a las víctimas para que revelen información confidencial A type of cyberattack in which attackers attempt to deceive victims into revealing confidential information
Software firewall Un dispositivo o software que controla el tráfico de red entrante y saliente A device or software that controls incoming and outgoing network traffic
Security audit Un examen sistemático de un sistema informático o red para identificar vulnerabilidades A systematic examination of a computer system or network to identify vulnerabilities
Software update Una versión mejorada de un software que corrige errores y vulnerabilidades An improved version of software that fixes errors and vulnerabilities
Security policy Un documento que define los principios y procedimientos de seguridad de una organización A document that defines the security principles and procedures of an organization


Programación y Desarrollo Web
Image Name Concepto Concept
HTML HyperText Markup Language, un lenguaje de marcado que se utiliza para crear páginas web HyperText Markup Language, a markup language used to create web pages
CSS Cascading Style Sheets, un lenguaje de hojas de estilo que se utiliza para definir la apariencia de las páginas web Cascading Style Sheets, a style sheet language used to define the appearance of web pages
JavaScript Un lenguaje de programación que se utiliza para agregar interactividad a las páginas web A programming language used to add interactivity to web pages
Python Un lenguaje de programación de propósito general que se utiliza para una amplia gama de aplicaciones, incluyendo el desarrollo web, el aprendizaje automático y el análisis de datos A general-purpose programming language used for a wide range of applications, including web development, machine learning, and data analysis
Java Un lenguaje de programación de código abierto que se utiliza para una amplia gama de aplicaciones, incluyendo el desarrollo web, las aplicaciones móviles y los juegos An open-source programming language used for a wide range of applications, including web development, mobile applications, and games
C++ Un lenguaje de programación de alto rendimiento que se utiliza para una amplia gama de aplicaciones, incluyendo sistemas operativos, juegos y aplicaciones de gráficos A high-performance programming language used for a wide range of applications, including operating systems, games, and graphics applications.
Framework Un conjunto de herramientas y bibliotecas que se utilizan para simplificar el desarrollo de aplicaciones A set of tools and libraries that are used to simplify the development of applications
API Application Programming Interface, un conjunto de reglas y especificaciones que permiten que dos aplicaciones se comuniquen entre sí Application Programming Interface, a set of rules and specifications that allow two applications to communicate with each other
MVC Model-View-Controller, un patrón de arquitectura de software que se utiliza para estructurar aplicaciones web Model-View-Controller, a software architecture pattern used to structure web applications
Frontend La parte de una aplicación web que interactúa con el usuario The part of a web application that interacts with the user
Backend La parte de una aplicación web que procesa datos y genera respuestas The part of a web application that processes data and generates responses
Database Una colección de datos organizada que se puede almacenar y recuperar electrónicamente A collection of organized data that can be electronically stored and retrieved
SQL Structured Query Language, un lenguaje de programación que se utiliza para interactuar con bases de datos Structured Query Language, a programming language used to interact with databases
